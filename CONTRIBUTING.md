# Contributing

This is a doctoral research instrument. The codebase is public as a commitment to methodological transparency, not as an invitation to feature contributions. That said:

**Bug reports** are welcome. Open an issue with reproduction steps.

**Security issues** should be reported privately to grady@nrgmr.com before public disclosure.

**Data corrections** — if you believe a score, provenance record, or evidence panel contains an error, use the public appeal system at /artifact/[id].

**Takedown requests** — if your content is in the corpus and you want it removed, use the form at /takedown.

**Code contributions** — the project is not accepting pull requests during the active dissertation period. After the dissertation is defended, contribution guidelines will be updated.

All contributions to content (notes, scoring reasoning, methodology text) are subject to the CC BY-NC 4.0 license in LICENSE-content.md.
